# Мониторинг модели рекомендаций

## Описание
Данный документ описывает метрики и логику мониторинга для системы рекомендаций банковских продуктов.

## Метрики
- **Precision@K**: Доля релевантных товаров в топ-K рекомендаций;
- **Recall@K**: Доля найденных релевантных товаров;
- **Data Drift**: PSI для распределения входных признаков.

## Отправка метрик
Метрики логируются в `logs/metrics.jsonl`.
